3 - Utilisation
Créez une div, un bouton ou peu importe avec la classe : "tipstools"
Ajoutez ensuite un attribut data-msg avec le message que vous voulez afficher
Ajoutez enfin un dernier attribut tips-pos qui correspond à la position dans le guide d'utilisateur. Par exemple cette étape est la numéro 3 donc j'ai utilisé l'attribut tips-pos="3". 
	    						
	    							< div class="col-md-12 tipstools" style="margin-top:50x;background:white;margin-top:20px;font-size:14px;"  tips-msg="Deuxième carré qui permet de montrer à l'utilisateur ou on se trouve :-)" tips-pos="3">
